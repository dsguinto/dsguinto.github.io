Wordpress Website:
The Tea Room - Bubble Tea Shop

The Tea Room is a made-up bubble tea shop website built using the Wordpress Content Management System.

This personal project was my introduction to Wordpress, where I implemented "from-scratch" development as opposed to working off of an existing theme. The layout, design, and back-end functionality were coded from scratch. Kitchen Beets is an application that utilizes the Spoonacular API and Youtube Data API.

When a user selects a recipe they would like to make, a Lofi music video is grabbed from Youtube and presented on the recipe page to give the user music to listen to while cooking. The length of the video (response: duration) will match or be within +10 minutes of the maximum amount of time (response: maxReadyTime) the recipe is expected to take to make. Currently, the application restricted by the number of API request available per day (150).

Skills Showcased: PHP, API Integration, HTML, CSS

HelpDesk:
Ticket Support System

For an XML course, I was required to create 2 XML files for a support ticket system: one for users and one for tickets. For each XML file, an XML Schema was created to define the structure for each file.

Once those XML files were created and populated with mock data, I was required to use PHP to create a functional support ticket system application using the data from the XML files. This included a login/logout function, displaying of tickets (user specific), ability to create a ticket, respond to tickets, and close/re-open exisiting tickets. PHP DOM methods were used to read and write to the XML files.

For the live demo, test login credentials can be found on github.

Skills Showcased: HTML, CSS, XML, PHP (DOM)
Github: github.com/dsguinto/HelpDeskApp
